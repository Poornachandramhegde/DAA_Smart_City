# Smart City Recovery System

A futuristic visualization-based DAA hackathon project built using React, Vite, and Tailwind CSS.

This project simulates synchronization recovery inside a distributed smart city control network using algorithmic techniques such as sorting, inversion analysis, dependency resolution, and graph-based activation sequencing.

---

## Features

- Smart City themed dashboard UI
- Station signal visualization
- ASCII-based station value conversion
- Corrupted vs restored ordering analysis
- Inversion count analysis
- Dependency matrix representation
- Topological sorting visualization
- Source removal dependency resolution
- Responsive futuristic interface

---

## Algorithms & Concepts Used

### Sorting & Disorder Analysis
- ASCII Signal Conversion
- Merge Sort Logic
- Inversion Count Analysis

### Dependency Resolution
- Directed Graph Representation
- Dependency Matrix
- Indegree Calculation
- Topological Sorting
- Kahn’s Algorithm (Source Removal Method)

## Getting Started

### Clone the Repository

```bash
git clone <repository-link>
```

### Navigate to Project Folder

```bash
cd smart-city-ui
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
